🎯 BẢNG 3: NGUYÊN TẮC VÀNG (Golden Rules to Remember)

┌─────────────────────────────────────────────────────────────────────────────────┐
│ RULE #1: UPTREND - GIÁ ↑ + CVD ↑ = LONG / DOWNTREND - GIÁ ↓ + CVD ↓ = SHORT │
│          Nếu CVD không follow giá → Xu hướng yếu → Scale out / Đợi xác nhận   │
├─────────────────────────────────────────────────────────────────────────────────┤
│ RULE #2: DIVERGENCE BULLISH (Giá Lower Low + CVD Higher Low) = HIGH PROBABILITY│
│          Nếu CVD tạo Higher Low 3+ lần = Đẳy đang hình thành → ChuẨN BỊ LONG  │
├─────────────────────────────────────────────────────────────────────────────────┤
│ RULE #3: DIVERGENCE BEARISH (Giá Higher High + CVD Lower High) = HIGH PROB     │
│          Nếu CVD tạo Lower High 3+ lần = Đỉnh đang tan rã → CHUẨN BỊ SHORT   │
├─────────────────────────────────────────────────────────────────────────────────┤
│ RULE #4: SIDEWAY + CVD TĂNG LIÊN TỤC = TÍCH LŨY = BREAKOUT TĂNG SẮP TỚI      │
│          Đặt pending order trên resistance → Chờ nến close outside → Vào follow │
├─────────────────────────────────────────────────────────────────────────────────┤
│ RULE #5: SIDEWAY + CVD GIẢM LIÊN TỤC = PHÂN PHỐI = BREAKDOWN GIẢM SẮP TỚI    │
│          Đặt pending order dưới support → Chờ nến close outside → Vào follow   │
├─────────────────────────────────────────────────────────────────────────────────┤
│ RULE #6: OI TĂNG = TREND MẠNH / OI GIẢM = TREND YẾU                            │
│          Nếu OI giảm đột ngột >10% → THANH LÝ → TRÁNH VÀO LỆNH MỚI           │
├─────────────────────────────────────────────────────────────────────────────────┤
│ RULE #7: FR QUỐC CAO (>0.1%) = CẢNH BÁO / FR ÂM (<-0.05%) = CƠ HỘI            │
│          Nếu FR >0.1% + LONG → Scale out 50%, có thể SHORT để arbitrage       │
├─────────────────────────────────────────────────────────────────────────────────┤
│ RULE #8: CVD + OI + GIỎ VỤ = TUYỆT ĐỐI NGỒI NGOÀI (Choppy market)             │
│          KHÔNG scalp trong choppy → Chỉ thua tiền mà thôi → Chờ 1-2 giờ       │
├─────────────────────────────────────────────────────────────────────────────────┤
│ RULE #9: LUÔN ĐẶT SL (Stop Loss) = LUÔN LUÔN = KHÔNG BƯỚC NGOẠI LỆ             │
│          SL = Bảo vệ vốn → Nếu không có SL = Gambling, không phải trading     │
├─────────────────────────────────────────────────────────────────────────────────┤
│ RULE #10: R/R PHẢI >= 1:2 (Tối thiểu) / 1:3-1:5 (Tốt nhất) = NGUYÊN TẮC SAO   │
│           Nếu R/R < 1:2 → BỎ QUA, chờ setup tốt hơn → Chất lượng > Số lượng   │
└─────────────────────────────────────────────────────────────────────────────────┘

💰 BẢNG 5: QUẢN LÝ VỐNS - TÍNH TOÁN POSITION SIZE (Dòng chảy tiền)

┌─────────────────────────────────────────────────────────────────────────────────┐
│ NẾU CÓ:  $1000 vốn   →  1 lệnh = 1-2% vốn = $10-$20 risk/lệnh                │
│ NẾU CÓ:  $10000 vốn  →  1 lệnh = 0.5-1% vốn = $50-$100 risk/lệnh             │
│ NẾU CÓ:  $100000 vốn →  1 lệnh = 0.5% vốn = $500 risk/lệnh (Chuyên nghiệp)   │
├─────────────────────────────────────────────────────────────────────────────────┤
│ CÔNG THỨC:                                                                      │
│  Position Size = (Vốn × Risk%) / (Giá - SL)                                   │
│                                                                                  │
│ VÍ DỤ:  $1000 vốn, Risk 1% = $10, BTC $40000, SL $39950 = ?                  │
│  Position Size = $10 / ($40000 - $39950) = $10 / $50 = 0.2 BTC (size nhỏ)   │
├─────────────────────────────────────────────────────────────────────────────────┤
│ PYRAMID / THÊM VỊ THẾ:                                                           │
│  Entry 1: 30% vốn (biggest risk)                                              │
│  Entry 2: 25% vốn (pullback xác nhận)                                         │
│  Entry 3: 20% vốn (pullback lần 2)                                            │
│  Remaining 25% = Reserve cho unexpected                                       │
├─────────────────────────────────────────────────────────────────────────────────┤
│ SCALING OUT (Cắt lỗi dần):                                                      │
│  Khi lỗ 10% TP → Close 33%                                                    │
│  Khi lỗ 50% TP → Close 33%                                                    │
│  Remaining 33% → Move SL break even                                           │
│  (= Take profit dần từ dưới lên, giảm rủi ro)                                 │
└─────────────────────────────────────────────────────────────────────────────────┘

│  □ 5. Nến xác nhận rồi chưa? (Pin Bar, Hammer, Engulfing?)                    │
│  □ 6. Volume cao không? (Gấp đôi volume trung bình?)                           │
│  □ 7. SL mình đặt ở đâu? (Dưới support? Trên resistance?)                     │
│  □ 8. R/R >= 1:2 không? (Nếu <1:2 → bỏ qua)                                  │
│  □ 9. Market có choppy không? (CVD loạn? → bỏ qua)                            │

⚠️ NHỮNG SAI LẦM CẦN TRÁNH (Fatal Mistakes)

 ❌ Trading choppy market → Thua tiền dễ (Không làm)
 ❌ Không có SL → Rủi ro vô hạn (PHẢI có SL)
 ❌ Size lớn quá → Tâm lý bị ảnh hưởng (Chỉ 1-2% vốn)
 ❌ Đế vào khi FR quốc cao → Liquidation sắp (Tránh)
 ❌ DCA vô tội → Lỗ càng nhiều (Chỉ DCA có xác nhận)
 ❌ Không quản lý rủi ro → Nợ vốn (PHẢI quản lý)
 ❌ Quá tham lam (Không chốt lãi) → Lỗ từ lãi (Chốt kịp thời)
 ❌ FOMO (Fear of Missing Out) → Vào lệnh sai (Kiên nhẫn)
